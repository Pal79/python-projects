# How to run

> enter app dir and create .venv:

```python3 -m venv .venv```

> activate .venv:

```source .venv/bin/activate```

> install flask:

```pip install flask```

> run app:

```python3 server.py```

> open web browser and run this:

```127.0.0.1:5001```

> stop app run:

```Ctrl+C```

> exit the venv:

```deactivate```
